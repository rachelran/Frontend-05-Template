# Yeoman   
### 安装yeoman   
`npm install -g yo`   

### 生成generator   
`yo generator`   

### npm link   
链接到全局执行环境，为npm包创建软链接   

# Webpack打包工具   
最初是为node服务的，多文件合并可以通过loader   
webpack-cli: 提供 webpack 命令,需要单独安装   

# Babel JS的编译工具    
### 安装   
`npm install -g @babel/core @babel/cli`   
### 配置文件：.babelrc   
`{          
  "presets": ["@babel/preset-env"]      
}`
